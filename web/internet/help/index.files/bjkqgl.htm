学员考勤管理

    学员考勤数据来源于学员上课刷卡记录。学员管理处或班主任可以查看、修改班级考勤的情况。考勤状态分为正常、迟到、缺课、公假、事假五项。

  1. 班级学员考勤状态显示。
  2. 班级学员考勤数据修改采用表格选择方式。
  3. 查询能以班级名称、学员姓名、课程名称、授课教师、时间段进行组合查询，支持模糊查询。
  4. 查询的结果要能导出到EXCEL表格中。（这部分移至表格输出）样表见表一、二、三。

    对查询的结果提供在线打印、预览功能。（这部分移至表格输出）
