公共事务

    公共事务管理是用户所处理的共有的事务管理。分为四个模块：工作安排，公告通知，共享文件夹，公共通讯录。

工作安排是部门负责人对自己部门人员或者其他人员指派工作安排，以及用户接受工作安排查看的过程。分为：接受工作安排和分派工作安排。

    接受工作安排是用户接受别人指派的工作安排，并查看后作出回复的过程。
分派工作安排是部门负责人对人员分派工作安排管理，分为增删改查管理。对已有回复的也可以作出回复。

    公告通知是用户对他人发出通知的管理，以及用户通知查看的过程。分为：查阅通知和公告通知管理。
查阅通知是用户接受通知，对通知查看

公告通知管理是通知发起人发出通知的管理，分为增删改查。

 

    共享文件夹是用户实现文件共享的功能，用户对文件上传和下载，并可以指定共享，查看文件的人员。

 

    用户查看公共通讯录的信息，由指定人员对公共通讯录作新增，修改，删除操作。
