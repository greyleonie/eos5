9•  教材管理

教材管理提供对教材的统一管理。教材管理包括：教材信息管理、教材订购管理、教材采购管理、教材发放管理、教材盘点管理、教材库存查询。

 

 

点击“教务管理 à 教材管理 ”，进入到 教材管理 页面。

 

9.1•  教材基本信息管理

功能： 对教材基本信息进行维护，包括增删改查基本功能。

操作： 点击“教材基本信息管理”菜单即可。

9.1.1•  查询

功能： 按班书名、作者、出版社进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。

9.1.2•  新增

功能： 新增教材基本信息。

操作： 点击“新增”按钮，弹出下面的新增页面。点击“新增”按钮可以增加一行输入栏，点“删除”按钮可以删除输入行。在对应的地方输入教材基本信息，按“保存”即可。

 

9.1.3•  修改

功能： 对已有教材基本信息进行修改。

操作： 选择要修改的行（可以多选）， 点击“修改”按钮，进入下面的修改页面，修改书本基本信息后，按“保存”即可。

 

 

 

9.1.4•  删除

功能： 删除已有的 书本基本信息 。

操作： 选择要删除的书本， 点击“删除”按钮，系统提示“是否真的要删除指定项目”，点击“确认”即删除。

 

9.1.5•  打印

功能： 对已有教材基本信息列表进行打印。

操作： 点击“打印“按钮，弹出下面的打印页面，可以进行打印、页面设置、预览操作。

9.1.6•  导出

功能： 将查询结果信息导出到 EXCEL 表格中 。

操作： 点击“导出“按钮，弹出下面的提示窗口，如选择”打开“直接 EXCEL 文件，选择“保存”则将 EXCEL 保存到本机上。

 

 

选择打开，则看到下边的导出的 EXCEL 文件，用户可以对该文件进行调整修改、打印。

 

 

 

 

 

9.2•  教材订购管理

功能： 教材订购是由教务处提出订购申请，领导进行审批。系统只需记录订购申请信息，不需要对审批流程进行管理。

操作： 点击“教材订购管理”菜单即可。

9.2.1•  查询

功能： 按年度，时期进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。

 

9.2.2•  新增

功能： 新增教材订购。

操作： 点击“新增”按钮，弹出下面的新增页面。点击“新增明细”按钮可以增加一行输入栏，点“删除明细”按钮可以删除输入行。把鼠标放到“书名”输入框，按下空格键可以弹出已有的书本。选择需要采购的书本，按“保存”即可。如果没有供选择的书本，则在书本基本信息管理中增加该书本信息。

 

9.2.3•  修改

功能： 对已有教材订购进行修改。

操作： 选择要修改的行， 点击“修改”按钮，进入下面的修改页面，可以点击“新增明细”按钮增加书本，也可点击“删除明细”来删除书本。可以修改书本的购数，修改书本订购后，按“保存”即可。

 

 

 

9.2.4•  删除

功能： 删除已有的书本订购。

操作： 选择要删除的书本订购单， 点击“删除”按钮，系统提示“是否真的要删除指定项目”，点击“确认”即删除。

9.2.5•  订购明细

功能： 查看订购单明细

操作： 选择要查看的书本订购单， 点击“订购明细”按钮即可。

 

9.3•  教材采购管理

功能： 记录教材采购入库信息，并可以查询。

操作： 点击“教材采购管理”菜单即可。

9.3.1•  查询

功能： 按书名，作者，经手人，入库时间，类别进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。

 

9.3.2•  新增

功能： 新增教材采购。

操作： 点击“新增”按钮，弹出下面的新增页面。点击“新增明细”按钮可以增加一行输入栏，点“删除明细”按钮可以删除输入行。把鼠标放到“书名”输入框，按下空格键可以弹出已有的书本。选择需已采购的书本，按“保存”即可。

 

•  订购单

功能： 选择采购单上的教材进行入库。

操作： 按“订购单”即可。

 

 

弹出采购单选择页面选择要入库的采购单，按“保存”按钮。

 

9.3,3•  修改

功能： 对已有教材采购进行修改。

操作： 选择要修改的行， 点击“修改”按钮，进入下面的修改页面。可以修改书本的购数，修改书本采购后，按“保存”即可。

 

 

 

9.3.4•  删除

功能： 删除已有的书本订购。

操作： 选择要删除的书本订购单， 点击“删除”按钮，系统提示“是否真的要删除指定项目”，点击“确认”即删除。

 

 

9.4•  教材发放管理

功能： 记录教材出库信息，并可以查询。

操作： 点击“教材发放管理”菜单即可。

9.4.1•  查询

功能： 按书名，班级，部门，发放时间进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。

 

9.4.2•  新增

功能： 新增教材发放。

操作： 点击“新增”按钮，弹出下面的新增页面。点击“新增明细”按钮可以增加一行输入栏，点“删除明细”按钮可以删除输入行。把鼠标放到“书名”输入框，按下空格键可以弹出已有的书本。选择需已发放的书本，按“保存”即可。

9.4.2.1•  订购单

功能： 选择发放单上的教材进行入库。

操作： 按“订购单”即可。

 

 

弹出发放单选择页面选择要出库的发放单，按“保存”按钮。

 

9.4.3•  修改

功能： 对已有教材发放进行修改。

操作： 选择要修改的行， 点击“修改”按钮，进入下面的修改页面。可以修改书本的购数，修改书本发放后，按“保存”即可。

 

 

 

 

9.4.4•  删除

功能： 删除已有的书本订购。

操作： 选择要删除的书本订购单， 点击“删除”按钮，系统提示“是否真的要删除指定项目”，点击“确认”即删除。

9.5•  教材盘点管理

功能： 记录教材盘点信息，并能将盘点库存与帐面库存进行对照，找出差额。

操作： 点击“教材盘点管理”菜单即可。

9.5.1•  查询

功能： 按盘点时间进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。

 

9.5.2•  新增

功能： 新增教材盘点。

操作： 点击“新增”按钮，弹出下面的新增页面。点击“新增明细”按钮可以增加一行输入栏，点“删除明细”按钮可以删除输入行。把鼠标放到“书名”输入框，按下空格键可以弹出已有的书本。选择需已盘点的书本，按“保存”即可。

 

9.5.3•  修改

功能： 对已有教材盘点进行修改。

操作： 选择要修改的行， 点击“修改”按钮，进入下面的修改页面。可以修改书本的入库、出库、差额、盘点库存，按“保存”即可。

 

 

 

 

9.5.4•  删除

功能： 删除已有的书本盘点。

操作： 选择要删除的书本盘点记录， 点击“删除”按钮，系统提示“是否真的要删除指定项目”，点击“确认”即删除。

9.6•  教材库存查询

功能： 记录 教材库存 信息，并能按照书名进行查询。

操作： 点击“ 教材库存查询 ”菜单即可。

9.6.1•  查询

功能： 按书名进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。
