学籍管理

学籍管理是对学员学籍资料的管理。学籍管理要实现学员学籍数据管理模板下载（EXCEL表格形式）、学员学籍数据录入（包括学员学籍数据批量录入［导入］和个别录入两种方式）、学员学籍数据修改（包括修改、增加、删除）、学员注册登记、退学登记等功能。

此部份的功能主要是学员学籍数据的管理，检索查询的功能移到学员信息检索

  • 学员学籍数据管理模板（EXCEL表格形式）下载。（表格样式见原系统）
  • 批量学员学籍数据（EXCEL表格格式管理模板）导入数据库，支持模糊录入。
  • 个别学员学籍数据（单个学员管理模板格式）录入数据库，支持模糊录入。
  • 学员学籍数据修改。
  • 表格选择式学员注册登记，未注册的学员系统自动删除。
  • 学号系统自动配置，学员学号的配置在学员注册程序确认后随机配给，配给后不可更改。
  • 表格选择式学员退学登记，选择确认后自动弹出学员学籍表进行填写，记录其退学的原因和退学时间。确认后该学员的党籍数据自动从本班学员数据中删除，自动进入退学学员数据库备查。可以根据学号、姓名、班级、退学时间查询退学记录，系统支持模糊查询（查询的功能移至学员信息检索）。
  • 班级学员的排列顺序号可任意调整、修改，要求系统具备自动排列功能。
