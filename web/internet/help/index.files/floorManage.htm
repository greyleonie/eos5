10•  场地信息

反映校课室、客房的使用或出租情况，教务处通过这样的一个查询，可以合理按排课室上课及对课室进行有效调整。

点击“教务管理 à 场地信息 ”，进入到 场地信息 页面。

 

10.1•  课室使用查询

功能： 对课室的使用情况进行查询。系统可以根据教学计划生成各课室占用情况一览表。包括是否占用（以不同颜色区分）、占用时间、班级、教学内容、授课教师。

操作： 点击“课室使用查询”菜单进入课室使用情况查询页面。

•  用鼠标悬浮在（上，下，晚）字上可以出现该课室的相关使用信息。

•  用鼠标悬浮在上栏课室名字上可以出现该课室的信息。

•  点击上栏的课室，在下栏可以显示当月的使用情况。

•  点击下栏的时间，在上栏可以出现所选时间的课室占用情况。

10.2•  客房使用查询

功能： 对客房的使用情况进行查询。系统可以根据住宿安排生成各客房占用情况一览表。包括是否占用（以不同颜色区分）、占用时间。

操作： 点击“客房使用查询”菜单进入客房使用情况查询页面。

•  用鼠标悬浮在上栏客房名字上可以出现该客房的信息。

•  点击上栏的客房，在下栏可以显示当月的使用情况。

•  点击下栏的时间，在上栏可以出现所选时间的客房占用情况。

 

10.3•  场地信息维护

功能： 对课室基础信息进行维护。

操作： 点击“场地信息维护”菜单即可。

 

 

 

10.3.1•  楼宇信息

功能： 对楼宇信息进行维护。

操作： 点击“楼宇信息”菜单即可。

有两种显示方式，列表 和 平铺。当点击平铺时以平铺方式显示

在平铺页面对应的楼宇上双击鼠标可以修改楼宇信息。

 

 

10.3.1.1•  查询

功能： 按楼宇名进行查询。

操作： 输入上面的查询条件，点击“查询”按钮即可。

 

10.3.1.2•  新增

功能： 新增楼宇信息。

操作： 点击“新增”按钮，弹出下面的新增页面。在对应的地方输入楼宇基本信息。楼宇名不能重复，按“保存”即可。

 

10.3.1.3•  修改

功能： 对已有楼宇信息进行修改。

操作： 选择要修改的行， 点击“修改”按钮，进入下面的修改页面，修改楼宇基本信息后，按“保存”即可。

 

 

 

10.3.1.4•  删除

功能： 删除已有的楼宇基本信息。

操作： 选择要删除的楼宇， 点击“删除”按钮，系统提示“是否真的要删除指定项目”，点击“确认”即删除。

 

10.3.1.5•  打印

功能： 对已有楼宇信息列表进行打印。

操作： 点击“打印“按钮，弹出下面的打印页面，可以进行打印、页面设置、预览操作。

10.3.1.6•  导出

功能： 将查询结果信息导出到 EXCEL 表格中 。

操作： 点击“导出“按钮，弹出下面的提示窗口，如选择”打开“直接 EXCEL 文件，选择“保存”则将 EXCEL 保存到本机上。

 

 

选择打开，则看到下边的导出的 EXCEL 文件，用户可以对该文件进行调整修改、打印。
