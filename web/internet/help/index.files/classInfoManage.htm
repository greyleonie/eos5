2  班级信息管理


班级信息管理包含对计划班级、当前班级和历年班级管理。其中当前班级管理要实现对当前班级信息的增、删、改、查功能，历年班级管理主要实现对历年班级管理的信息查询统计功能。可以将计划班级转换为当前班级或者将当前班级转换为历史班级。

点击“教务管理 à 班级信息管理”，进入到班级信息管理页面。

2.1  计划班级

功能： 对计划班级进行查询、新增、修改、删除、设为当前班级、导出、打印操作。

操作： 点击“教务管理 à 班级信息管理 à 计划班级”，进入到班级信息管理页面。

2.1.1•  查询

功能： 按班级类型、开班时间、调训单位、班级。

操作： 输入上面的查询条件，点击“查询”按钮即可。

2.1.2•  新增

功能： 新增计划班级。

操作： 点击“新增”按钮，进入下面的新增页面，输入班级信息，按“保存”即可。

2.1.3•  修改

功能： 对已有班级信息进行修改。

操作： 点击“修改”按钮，进入下面的修改页面，修改班级信息后，按“保存”即可。

2.1.4•  删除

功能： 删除已有的班级信息。

操作： 选择要删除的班级， 点击“删除”按钮，系统提示“是否真的要删除该班级”，点击“确认”即删除。

2.1.5•  设为当前班级

功能： 把计划班级设置为当前班级

操作： 选择班级后，点“设为当前班级” 按钮，系统提示“是否真的要删除该班级”，点击“确认”即把 计划班级设置当前班级 。

 

2.2•  当前班级

2.2.1•  查询

功能： 按班级类型、开班时间、调训单位、班级。

操作： 与计划班级的操作相同，可参照上面。

2.2.2•  新增

功能： 新增计划班级。

操作： 与计划班级的新增相同，可参照上面。

2.2.3•  修改

功能： 对当前班级信息进行修改。

操作： 与计划班级的修改相同，可参照上面。

2.2.4•  删除

功能： 删除当前班级信息。

操作： 与计划班级的删除相同，可参照上面。

2.2.5•  课程

功能： 查询班级开设的课程

操作： 选择班级后，点“课程” 按钮，弹出页面，显示所开设的课程

2.2.6•  学员名册

功能： 查询班级的学员

操作： 选择班级后，点“学员名册” 按钮，显示该班级的学员

 

2.2.7•  结束班级

功能： 把当前班级设置为历史班级

操作： 选择班级后，点“结束班级” 按钮， 系统提示“设置为历史班级，确认吗？”，点击“确认”即把当前班级转到历年班级。

2.3•  历年班级

2.3.1•  查询

功能： 按班级类型、开班时间、调训单位、班级。

操作： 与计划班级的操作相同，可参照上面。

2.3.2•  课程

功能： 查询班级开设的课程

操作： 与当前班级的课程相同，可参照上面

2.3.3•  学员名册

功能： 查询班级的学员

操作： 与当前班级的学员名册相同，可参照上面

2.3.4•  设为当前班级

功能： 把历年班级设置为当前班级

操作： 选择班级后，点“设为当前班级” 按钮， 系统提示“设置为历史班级，确认吗？”，点击“确认”即把当前班级转到历年班级。
