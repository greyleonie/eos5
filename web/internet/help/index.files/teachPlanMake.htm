4•  教学计划编制

4.1•  教学计划框架

4.1.1•  新增框架

功能： 新增教学框架

操作： 点击“新增框架”按钮，进入下面的新增页面，输入框架信息，按“保存”即可。

4.1.2•  修改框架

功能： 修改教学框架

操作： 点击“修改框架”按钮，进入下面的修改页面，修改教学框架信息后，按“保存”即可。

4.1.3•  删除框架

功能： 删除已有的教学框架。

操作： 选择要删除的教学框架， 点击“删除框架”按钮，系统提示“框架相关的单元及课程同时删除，确定吗”，点击“确认”即删除

4.1.4•  编辑单元

功能： 编辑教学计划框的单元，包括添加、修改、删除单元；以及为单元添加、修改、删除课程

操作： 选择要教学框架， 点击“编辑单元”按钮，进入编辑单元页面

4.1.4.1•  添加新教学单元

功能： 添加新的教学单元

操作： 点击“添加新教学单元”按钮，进入下面的修改页面，输入单元信息 后，按“保存”即可，在保存该单元的同时，系统根据单元的开始和结束时间，生成课程的日期表，通过修改课程，可以完善课程的详细内容。

4.1.4.2•  修改单元

功能： 修改已有的教学单元

操作： 点击黄色的“修改”按钮，进入下面的修改页面，修改单元信息 后，按“保存”即可。

4.1.4.3•  删除单元

功能： 删除已有的单元

操作： 在对应的单元上面，点“删除”按钮，系统提示“删除该单元会删除其相关的课程，是否确定”， 点击“确认”即删除

4.1.4.4•  新建课程

功能 ： 为单元新增课程

操作： 点击“新增课程”按钮，进入下面的新增页面，输入课程信息，按“保存”即可

4.1.4.5•  修改课程

功能： 修改已有的课程

操作： 点击相应课程的“修改”，进入下面的修改页面，修改课程信息后，按“保存”即可。

4.1.4.6•  删除课程

功能：删除已有的课程

操作： 选择要删除的课程， 点击“删除”操作，系统提示“是否删除指定记录”，点击“确认”即删除

4.1.4.7•  提交领导审批

功能： 把教学计划提交给领导审核

操作： 点击“提交领导审批”按钮，出现选择领导页面，选择领导后提交就完成操作，教学计划提交到领导审核

4.1.5•  定为终稿

功能： 不经任何审批，直接把教学框架定为终稿

操作： 选择要定为终稿的教学框架， 点击“ 定为终稿 ”按钮，系统提示“确实定为终稿吗”，点击“确认”即完成

4.1.6•  查看框架

功能： 查看现有框架的详细信息

操作： 点击“查看框架”按钮，进入下面的详细的察看页面。

4.1.7•  授权

功能： 给某个教研部授权，能够修改课程的一些内容

操作： 选择要授权的教学计划， 点击“授权”操作，出现页面显示了该教学计划所有的单元，针对单元选择相应的 教研部，然后保存，完成授权。

4.2•  征求意见稿

4.2.1•  主体班教学计划

4.2.1.1•  加注意见

操作： 针对某个课程，点操作下的图标 ，出现填写意见的页面，输入意见后，点“保存”按钮即完成。

4.2.1.2•  意见汇总

操作： 点“意见汇总”按钮，出现查看意见的页面

操作： 点“导出”按钮，提示是直接打开还是保存到本地

4.2.2.2•  打印

功能： 打印教学计划

操作： 点“打印”按钮

 

4.2.2.3•  提交领导审批

功能： 把教学计划提交给领导审批

操作： 点“提交领导审批”按钮，出现选择领导页面，选择人员后，点“保存”即可

4.2.2.4•  定为拟开专题菜单

功能： 把教学计划定为终稿

操作： 点“定为拟开专题菜单”按钮即可，系统提示“操作成功！”

4.2.2.5•  发送到外网

功能： 把教学计划发送到外网

操作： 点“发送到外网”按钮，出现预览页面，然后点“保存”按钮即可

 

4.3•  课题筛选

4.3.1•  主体教学计划

4.3.1.1•  采纳课程

功能： 把课程设置为已采纳状态

操作： 在每个课程对应的“是否被采纳”下打勾表示采纳，选择采纳的课程后，点“保 存”按钮即完成

4.3.1.2•  定为初稿

功能： 把教学计划定为初稿，把教学计划提交给领导审批。

操作： 点“定为初稿”按钮，出现选择页面，选择审批领导后，点“保存”按钮即完成。

4.3.1.3•  定为终稿

功能： 把教学计划定为终稿

操作： 点“定为初稿”按钮，系统提示“确实要定为终稿吗！”，点“确定”按钮，系统提示“操作成功！”，完成该操作，该教学计划在“教学计划定稿”模块可以看到。

4.3.1.4•  意见汇总

功能： 查看领导在审核时，对教学计划加注的意见

操作： 点“意见汇总”按钮，出现查看页面

4.3.1.5•  已采纳课程

功能： 列出所有已采纳的课程

操作： 点“已采纳课程”按钮，页面刷新一次，列出已采纳的课程

4.3.1.6•  所有课程

功能： 列出所有的课程

操作： 点“所有课程”按钮，页面刷新一次，列出所有的课程

4.4.1•  待审批教学计划

操作： 点“汇总审批意见”按钮，出现查看页面，在该页面，上部分列出所有针对课程的意见，下部份可以输入针对整个计划的意见，点“保存”按钮，当前审批意见被保存下来。

4.4.1.4•  加注审批意见

功能： 给每个课程加注意见

操作： 点每个课程所在行的图标，出现填写页面，输入意见后，保存即可。

 

4.4.2•  已审批教学计划

功能： 查看本人审核或者审批过的教学计划

操作： 通过下拉列表可以选择查看其他班级的计划
