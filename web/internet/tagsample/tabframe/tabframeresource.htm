标签书写方式：

<eos:tabframe id="test" width="100%" height="100%" frameborder="0" scrolling="auto">
    <eos:tabframetitle url="../testhtml/test1.htm" property="urls/url1"type="xpath" title="仓储物流"/>
    <eos:tabframetitle url="../testhtml/test2.htm" selected="true" title="仓储物流"/>
    <eos:tabframetitle url="../testhtml/test3.htm" property="urls/url2"type="xpath" title="工位器具设备"/>
</eos:tabframe>

对应的Entity:

<root>
  <data>
    <urls>
      <url1>
        <id>10</id>
        <name>wang</name>
      </url1>
      <url2>
        <id>100</id>
        <name>li</name>
      </url2>
    </urls>
  </data>
</root>

标签详细说明：

 

Tag名称

说明

< eos:tabframe />

TabFrame主标签

< eos:tabframetitle />

Tab标签

 

< eos:tabframe />的属性

说明

id

标识TabFrame，同时生成的Iframe的名称为id后面添加 _frame。

width

生成Iframe的宽度，可以为百分比

height

生成Iframe的高度，可以为百分比

frameborder

生成Iframe的边框宽度，设为0表示无边框

scrolling

生成Iframe是否出现滚动条，可以为auto,no

 

<eos:tabframetitle />的属性

说明

url

标签对应的URL

property

URL对应的对数的Xpath，type为xpath时有效

title

标签显示的值

selected

初始时是否选中本TAB,取值为true 或 false.

type

生成URL的类型，如果为xpath，则URL中要添加property参数对应的entity中的值对。如URL?parametername=parametervalue的形式。
