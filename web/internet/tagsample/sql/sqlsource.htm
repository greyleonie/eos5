标签书写方式：

表示层设计：

Tag名称

说明

< eos:enum />

对resultset进行便利

<eos:field/>

每个数据的显示

 

< eos:enum />的属性

说明

packageName

获取数据连接的包名

id

标识此查询的ID，默认为SQL_Tag

querySQL

设置进行查询的select语句，

 

<eos:field />的属性

说明

id

标识此查询的ID，默认为SQL_Tag,同如果自己建立，则要同sql:enum中的id相同。

maxLength

显示的field的最大宽度

property

对应要显示的field名称，在querySQL语句中应包含此字段。
