标签书写方式：

Entitytable属性定义

说明

property

Xpath，对应Entity中的数据

labelCss

Label 的css，标签的说明

bodyCss

Body的CSS，内容

styleClass

HTML属性，Table的CSS

columnNum

表的列数，必须是2的倍数

width

HTML属性，Table的宽度

height

HTML属性，Table的高度

align

HTML属性，Table的对齐方式

border

HTML属性，Table的边框宽度

Id

Table的标识

 

如：

<eos:entitytable id="a" name="test" property="list" columnNum="4" labelCss="ss" bodyCss="ss"/>

对应的显示类型支持：textarea　select radio text checkbox

对应的Entity:

<list>

          <person>

             <name>person/name</name>

             <label>姓名</label>

             <value>张三</value>

             <type>text</type>

          </person>

          <person>

             <name>person/occupation</name><!―表示数据的名称à

             <label>性别</label><!―表示数据的提示à

             <value>1</value><!―表示数据的值à

             <type>radio</type><!―表示数据的显示方式à

             <dataMode>biz</dataMode><!―表示数据字典à

             <bizType>BNDICT_gender</bizType><!―表示数据字典的类型à

          </person>

</list>

本示例对应的代码如下：

<%@ include file="../head.jsp"%>
<eos:entitytable name="test" property="list" columnNum="4" labelCss="ss" bodyCss="ss" />
