标签书写方式：

Tab Tag

说明

<eos:tab>

Tab标签

<eos:tabtitle>

Tab的标题条目

<eos:titleitem >

Tab的标题内容

<eos:tabcontent>

Tab的内容条目

<eos:contentitem>

Tab的内容

 

 

Tab属性定义

说明

id

Tab的标识

height

Tab的高度

width

Tab的宽度

imgPath

Tab所用图片的路径，默认为　/internet/images

tabtitle属性定义

无

titleitem属性定义

包含在tabtitle中

selected

TAB是否选中

tabcontent属性定义

无

contentitem属性定义

无
