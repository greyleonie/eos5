webFrame标签说明

上下结构：

<eos:webframe type="top_bottom" template="template.jsp">

       <eos:webframeitem location="top" width="100%" height="80%" frameborder="0" scrolling="auto" url="menu.htm"   />

    <eos:webframeitem location="bottom" width="100%" height="100%" frameborder="0" scrolling="auto" url="a.htm"   />

</eos:webframe>

左右结构：

<eos:webframe type=" top_left_righ " >

       <eos:webframeitem location="top" width="100%" height="80%" frameborder="0" scrolling="auto" url="logo.htm"   />

    <eos:webframeitem location="left" width="100%" height="100%" frameborder="0" scrolling="auto" url="a.htm"   />

    <eos:webframeitem location="right" width="100%" height="100%" frameborder="0" scrolling="auto" url="b.htm"   />

</eos:webframe>

表示层设计：

Tag名称

说明

< eos: webframe />

webframe主标签

< eos: webframeitem />

Webframe条目标签

 

< eos: webframe />的属性

无

type

Webframe的类型，可以为top_bottom或者top_left_right,对应的框架类型为上下结构，上－左右结构，现仅支持这两种结构

template

webframe对应的模板文件，可以自己重写，以适应新的布局。其默认值为：

如type=top_bottom，则template=/internet/template/ top_bottom.jsp

type=top_left_right，则template=/internet/template/ top_left_right.jsp

 

<eos: webframeitem />的属性

说明

width

生成Iframe的宽度，可以为百分比

height

生成Iframe的高度，可以为百分比

frameborder

生成Iframe的边框宽度，设为0表示无边框

scrolling

生成Iframe是否出现滚动条，可以为auto,no

location

该框架条目在框架中的位置，必须为top bottom对应type=top_bottom，或者top　left　right　对应type=top_left_righ，如果其值为top，则width等属性不起作用，因为上部不是用 frame形成，而是直接加入，比如MENU等。

本示例代码：

上下结构：

<%@ page language="java" contentType="text/html; charset=GB2312"%>

<%@ taglib uri="/WEB-INF/hci-eos.tld" prefix="eos" %>


<eos:webframe type="top_bottom" template="template_top_bottom.jsp">
<eos:webframeitem location="top" width="100%" height="80%" frameBorder="0" scrolling="auto" url="./top/menu.jsp" />
<eos:webframeitem location="bottom" width="100%" height="400" frameBorder="0" scrolling="auto" url="../testhtml/test1.htm" />
</eos:webframe>

 

左右结构：

<%@ page language="java" contentType="text/html; charset=GB2312"%>

<%@ taglib uri="/WEB-INF/hci-eos.tld" prefix="eos" %>


<eos:webframe type="top_bottom" template="template_top_left_right.jsp">
<eos:webframeitem location="top" width="100%" height="80%" frameBorder="0" scrolling="auto" url="./logo.jsp" />
<eos:webframeitem location="left" width="100%" height="100%" frameBorder="0" scrolling="auto" url="./left/index.jsp" />
<eos:webframeitem location="right" width="100%" height="100%" frameBorder="0" scrolling="auto" url="../testhtml/test1.htm" />
</eos:webframe>

对应的模板文件：

template_top_bottom.jsp

<%@ page language="java" contentType="text/html;charset=GB2312"%>
<%@ taglib uri="/WEB-INF/hci-eos.tld" prefix="eos" %>
<html>
<body>

<eos:disp name="top"/>

<eos:disp name="bottom"/>

</body>
</html>

template_top_left_right.jsp

<%@ page language="java" contentType="text/html;charset=GB2312"%>
<%@ taglib uri="/WEB-INF/hci-eos.tld" prefix="eos" %>
<html>
<body>
<table width="100%"><tr width="100%">
<td colspan="2">
<eos:disp name="top"/>
</td>
</tr>
<tr>
<td height="400" width="150">
<eos:disp name="left"/>
</td>
<td height="400" width="600">
<eos:disp name="right"/>
</td></tr>
</table>
</body>
</html>
