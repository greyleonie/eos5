标签书写方式：

 

Tag名称

说明

<display:table/>

封装TABLE

<display:column/>

封装column

<display:param/>

封装超链接属性

 

<display:table/>的属性设计

说明

property

要显示的list的xpath

pageUrl

分页时用的URL，一般同当前页的URL相同

styleClass

表格的CSS，生成如下：<table class=’’>
style 表格的CSS，生成如下：<table style=''>
border 表格连框的宽度，默认为0
width 表格的宽度
background
表格的图片
bgcolor 表格的背景颜色
height 表格的高度
cellpadding 表格的cellpadding
cellspacing 表格的cellspacing

id

Table的id

 

<display:column/>：封装html:column属性

<display:column/>的属性设计

说明

type

类型（当前支持checkbox/input）

property

要显示的list的xpath

styleClass

表格中列的CSS，生成如下：<td class=’’>

title

本列的title

headerClass

设置列头的css, class=’’

sortable

是否可以排序，可以为true或false

staticUrl

列以静态显示的url
align 表格列的对齐方式，尽量用styleClass进行控制

url

Xpath，以xpath的内容作为URL，如果同时有staticUrl，则staticUrl优先

 

<display：param/>标签：必须在display：column内使用，而且在display:column的staticUrl或url属性不为空时有意义，用来设置超链接的参数

<display：param/>的属性设计

说明:

property

属性的xpath，最后在形成的url上加property=’propertyValue’

如：

<display:table id="test" property="list[@name='table']" pageUrl="display.jsp">

    <display:column property="index" sortable="true" title="序号"/>

    <display:column property="Customer/CustID" title="选项2" type="checkbox"  />

    <display:column property="Customer/CustName" title="名称" staticUrl="a.htm" sortable="true">

            <display:param property="Customer/CustID"/>

    </display:column>

   

    <display:column property="Customer/CustName" title="明细" url="Customer/url" sortable="true">

        <display:param property="Customer/CustID"/>

    </display:column>

   

    <display:column property="Customer/phone"  title="电话" />

    <display:column property="Customer/sex"  title="性别" type="checkbox"/>

 

   

</display:table>

注：如果entity中要显示的列中有属性dataMode="biz"和bizType ，是对应的显示值应从数据字典中取得其对应的值，如有多个中间用逗号分隔。
Eg：entity中有<sex dataMode="biz" bizType="BNDICT_gender">1</sex>，则显示　男
