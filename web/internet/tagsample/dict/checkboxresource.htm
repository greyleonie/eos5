标签书写方式：

 

tag定义

说明

<dict: multicheckbox />

封装业务字典的checkbox

属性

 

businTypeId

业务字典类别，可以没有，确定需要翻译业务代码的类别。

typeProperty

业务字典类别所存放DOM数据的XPATH，可以没有。注意：如果使用了typeProperty，businTypeId就不起作用，typeProperty和businTypeId必须有一个

property

可以没有，业务代码对应DOM数据中xpath

value

多选项值，可以没有，使用value属性，就是直接翻译常量代码。注意：如果使用了property，value就不起作用

id

可以配合循环使用的标签属性

accesskey

多选的checkbox具有的访问键属性

disabled

多选的checkbox具有的使无效属性

indexed

可以配合循环使用的序列属性

multiple

多选的checkbox具有的可多选属性

 

示例：

<dict:multicheckbox modifiable="true" name="wf" value="2,3,4" busintypeId="BNDICT_certType" formName="test"/>
