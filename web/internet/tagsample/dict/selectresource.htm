标签书写方式：

 

tag定义

说明

<dict:select/>

封装业务字典的select

属性

 

businessId

默认值

multiple

是否允许多选

size

多选时显示的大小

businTypeId

业务字典类型

privilegeCriteria

比较标识符，可以为> = < >= <= !=

privilegeValue

要比较的值

nullOption

是否在select中增加一提示条目，为true或false

nullLabel

增加提示条目的内容

property

Xpath，其值对应显示条目的默认值，

name

生成select的名称

typeProperty

Xpath, 对应业务字典类型

accesskey

Select列表框具有的访问键属性

disabled

Select列表框具有的使无效属性

indexed

可以配合循环使用的序列属性

multiple

Select列表框具有的可多选属性

onblur

Select列表框具有的激活事件

onchange

Select列表框具有的改变事件

onclick

Select列表框具有的鼠标单击事件

ondblclick

Select列表框具有的鼠标双击事件

onfocus

Select列表框具有的获得焦点事件

onkeydown

Select列表框具有的keydown事件

onkeypress

Select列表框具有的keypress事件

onkeyup

Select列表框具有的keyup事件

onmousedown

Select列表框具有的鼠标按下事件

onmousemove

Select列表框具有的鼠标移动事件

onmouseout

Select列表框具有的鼠标移出事件

onmouseover

Select列表框具有的mouseover事件

onmouseup

Select列表框具有的mouseup事件

style

Select的样式

styleClass

Select使用的样式类

size

Select列表框具有的大小

privilegeValue

privilegeValue可以没有，对应上面判断条件的值

privilegeCriteria

可以没有，privilege值的条件，参照BNDICT包中对业务字典表的说明，属性值只能是 "="，">"，"<"，">="，"<="，"!="，"<>"

 

示例：

<dict:select name="zhiye" businTypeId="BNDICT_certType"  businessId="2" />
